#Prototype pipeline
These are the available gulp tasks that live inside the honest prototype.


##Build tasks

###build-html
Build only HTML files

###build-js
Build only JS files

###build-sass
Build only SASS files

###build
Build all


##Watch tasks

###watch-html
Watch only HTML files

###watch-js
Watch only JS files

###watch-sass
Watch only SASS files

###watch
Watch everything

###sprite
Build spritessheet
